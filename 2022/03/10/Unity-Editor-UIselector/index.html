Scene UI Selection Filter

发表于 2022-03-10 更新于 2022-09-04 分类于 Unity ， Editor

功能

Unity中可以通过Scene窗口点击选择模型，但是UI之间是重叠关系，每次点击的时候选中的都是最前面的，即便这个UI并没有显示的内容。所以这里实现了一个UI过滤器，可以非常方便的快速选中看到的UI。
